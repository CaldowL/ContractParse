面试实操题：合同信息提取应用开发

### 背景
你所在的公司需要处理大量通信场地租赁/管理类合同。现有一批合同文档（文本格式），需要从中提取关键字段并结构化输出。请你**自行设计**提取逻辑（可以使用 LLM 提示词工程、规则+正则、或者混合方法），开发一个**合同信息提取应用**。应用可以基于 **Dify** 工作流，也可以用 **Python** 编写独立程序。最终需要能够接收一份合同文本（或文件），输出符合要求的 JSON 数组。

### 提取字段定义
你需要从合同文本中提取以下 8 个字段，字段含义和规则如下：

| 字段 | 提取规则说明 |
|------|--------------|
| **合同编号** | 优先从合同正文中“合同编号”“协议号”等字段提取；若正文未标注，则以文件名为编号。编号格式示例：`CU12-3456-7890-123456`（数字和短横线） |
| **合同类型** | 根据合同内容判断：<br>- 若租用场地作为通信设备机房 → `通信机房场地租用合同`<br>- 若涉及“甲方系国务院批准成立的，经营电信基本业务的大型国有企业...乙方愿意将产权（或管理权）属于自己的房屋及场地提供给甲方管理”类似描述 → `基站房屋及场地管理合同`<br>- 若安装无线覆盖系统 → `无线覆盖系统安装合同`<br>- 若为续签协议 → `续签合同`<br>- 若为主体变更协议 → `主体变更合同`<br>- 无法判断 → `其他合同` |
| **地址** | 提取合同中明确的场地地址。优先从“场地位于”“甲方同意乙方在...”等关键词后提取。若存在附表，从附表中提取。若未明确标注，可根据甲方注册地推断并标注“（推测）”。 |
| **金额** | 提取“物业管理费”“租金”“服务费”等关键词后的数值（如 `8000元/年`）。需注明**是否含税**、**支付周期**（年/月/日）及**币种**。 |
| **经度** | 若合同直接标注经纬度（如 `118.61696`），直接提取；未标注则留空字符串。 |
| **纬度** | 同上。 |
| **面积** | 提取“建筑面积”“使用面积”“场地面积”等关键词后的数值及单位（如 `30平方米`）；未标注留空。 |
| **租期** | 提取起止日期，格式 `YYYY/MM/DD-YYYY/MM/DD`。例如“期限自2014年9月5日至2024年9月4日” → `2014/9/5-2024/9/4`。若为固定年限（如“租期15年”），需结合合同签署日期计算。 |
| **备注** | 补充说明：<br>- 若合同有自动续期条款（“协议到期双方无异议继续生效”） → 备注“合同续期”<br>- 若为主体变更协议 → 备注“主体变更协议”<br>- 若字段模糊或格式复杂 → 备注“需人工复核”<br>- 其他特殊情况自由备注 |

### 输出格式
无论输入是单份合同还是多份合同，输出均为 **JSON 数组**，每个元素是一个对象，包含上述 8 个字段（顺序不限）。示例：
```json
[
  {
    "合同编号": "CU12-3456-7890-123456",
    "合同类型": "基站房屋及场地管理合同",
    "地址": "南京市浦口区江浦街道中圣北街20号",
    "金额": "25000元/年（含税，人民币）",
    "经度": "118.61696",
    "纬度": "32.06180",
    "面积": "30平方米",
    "租期": "2014/9/5-2024/9/4",
    "备注": "合同续期"
  }
]
```

### 技术要求
1. **输入**：支持直接传入合同文本字符串（简化处理，也可支持 `.txt` 文件上传）。
2. **输出**：严格输出上述 JSON 数组。
3. **实现方式**（二选一）：
   - **Dify 工作流**：设计工作流（可使用 LLM 节点、代码节点、HTTP 请求等），导出 DSL 文件或提供截图+说明。
   - **Python 程序**：调用任意 LLM API（必须兼容 OpenAI 接口格式），并**预留配置项**（模型名称、API Key、Base URL 可通过环境变量或配置文件修改）。
4. **扩展能力**（加分项）：
   - 集成 **MCP**（Model Context Protocol）工具，例如调用地图 API 对未提供经纬度的地址自动补全经纬度。
   - 设计一个 **SKILL**（如 LangChain Tool）处理“附表中的地址”或批量合同。
   - 增加日志、异常重试、输出格式校验。

### 提交内容

### 提示

- 考虑处理字段缺失、格式不规范等情况。
- 可以调用外部地图 API 实现“地址→经纬度”作为 MCP 工具演示。
